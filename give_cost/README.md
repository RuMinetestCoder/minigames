# give_cost

Give cost for:

* Crafting;
* Digging;
* Building;
* Chat messages;
* Per minute.

Config path: %worldpath%/configs/give_cost/settings.conf

Formula: setting number * booster

## Commands and privs

* **/gcreload** - reload settings (need *gcreload* priv)

## API

* *give_cost.set_boost(num)*
* *give_cost.get_boost()*

## Depends

* mod_configs
* coins
* initlib?
